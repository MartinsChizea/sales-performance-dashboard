# Sales Performance Dashboard

An interactive Excel dashboard analyzing sales performance across regions, 
products, and sales reps from January to May 2025.

## Tools Used
- Microsoft Excel (Pivot Tables, Charts, Slicers)
- Custom KPI Cards
- Data Visualization (Bar, Pie, Line Charts)

## Key Insights
- Total Revenue: ₦2.33B | Profit: ₦465.67M
- Laptop A13 was the highest-profit product at ₦105.34M
- Peter was the top-performing sales rep with ₦434.81M in profit
- Lagos had the highest COGS at ₦493.81M
- Customer count dropped sharply in May (536 → 18), worth investigating
- Abuja, Kano, and Port Harcourt contributed significantly to COGS
